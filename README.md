This project aims to model the short-term risk of the US equity market
using a Monte Carlo simulation on the SPY ETF (proxy for the S&P 500).

## Key features
- Historical volatility estimation from daily returns
- Monte Carlo simulation using a Geometric Brownian Motion
- 20-day horizon risk analysis (VaR, Expected Shortfall)

## Tools
- Python
- NumPy, Pandas, Matplotlib
- Yahoo Finance (public data)
